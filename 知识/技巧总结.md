# 总结  

## 数组拷贝

```
var arr = [1,2,3,4]
var clone = arr.slice()
```

## 获取数组最大/最小值

```
let arr = [1,2,3,4]
var max = Math.max.apply(null, arr )
var min = Math.min.apply(null, arr )
```

## 判断值类型

```
let arr = [1,2,3,4]
let type = Object.prototype.toString.call(arr)
```

## new Date()转化为时间戳

```
+new Date()
new Date() + 1 
```

```
let arr = [1,2,3,4]

```

## 中文名排序
```
 function sortChinese (arr) { // 参数： 排序的数组
    arr.sort(function (item1, item2) {
      return item1.localeCompare(item2, 'zh-CN');
    })
  }
  ```
